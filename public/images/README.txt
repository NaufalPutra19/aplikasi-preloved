INSTRUCTIONS:
=============
1. Upload your logo image file here and name it "logo.png"
2. Recommended size: height around 40px for navbar, 35px for sidebar
3. Supported formats: PNG, JPG, SVG (PNG recommended for transparent background)
4. The logo will automatically appear in:
   - Navbar (landing page)
   - Sidebar (admin dashboard)

NOTE: If you want to use a different filename, update the logo path in:
- resources/views/partials/navbar.blade.php
- resources/views/layouts/admin.blade.php
